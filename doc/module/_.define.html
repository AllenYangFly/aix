_.define

API:

_.define(obj,prop,setting)
_.define(obj,setting)

参数:

obj [ 类型:Object ]
prop [ 类型:String , Number ]
setting [ 类型:Object ]

说明:

_.define 用于给指定的Object对象定义属性,并且可以设置该属性的参数

示例:

1. _.define 可以定义单个属性，也可以一次性定义多个属性 
var a = {};

_.define(a,"a",{
  value : 1,
  enumerable: false,
  configurable: false,
  writable: true
});

console.log(a.a);

//定义多个属性,b和c
_.define(a,{
  "b":{
    value : 2,
    enumerable: true,
    configurable: true,
    writable: true
  },
  "c":{
    value : 3,
    enumerable: true,
    configurable: true,
    writable: true
  },
});

console.log(a.b,a.c);
输出结果:
>  1
>  2,3

2.定义属性时,设置enumerable configurable writable 
var a = {};

_.define(a,"a",{
  value : 1,
  enumerable: false,
  configurable: false,
  writable: false
});

console.log(a.a===1);

//无法设置a.a的数值，因为writable定义为false
a.a = 4;
console.log(a.a);
输出结果:
>  true
>  1
