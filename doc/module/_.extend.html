_.extend

API:

_.extend(proto,extendobj,notThisproperty)

参数:

proto [ 类型:Object ]
extendobj [ 类型:Object ]
notThisproperty [ 类型:String,Array ]

说明:

_.extend 函数有两个作用.

1. 在 _ (工具库) 上扩展自定义方法

2. 合并对象，并且可以筛选不需要附加的属性

_.extend 底层调用了_.compose

示例:

1. 在 _ 扩展方法，使其可以随处调用 
_.extend({
  hello: function(data){
    console.log("hello "+data);
  }
});

_.hello("aix");
输出结果:
>  "hello aix"

2. 合并b对象的所有属性到a对象 
var a = { a:1 , c:3 };
var b = { a:2 , b:1 };

_.extend(a,b);

console.log(a);
console.log(b);
输出结果:
>  "{ a:2,c:3,b:1 }"
>  "{ a:2,b:1 }"

3. 合并b对象的所有属性到a对象 . 并且忽略b对象上的a属性 
var a = { c:3 };
var b = { a:2 , b:1 };

_.extend(a,b,"a"); 

//合并时忽略了b中的a属性
console.log(a);
console.log(b);
输出结果:
>  "{ c:3,b:1 }"
>  "{ a:2,b:1 }"
