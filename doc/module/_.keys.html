_.keys

API:

_.keys(varible)

参数:

varible [ 类型:非null类型 ]

返回值:

[key,key1,...] [ 包含参数对象或者数组中所有的key或index的数组 ]

说明:

_.keys 整合出对象和键，或者数组的下标(index)，作为数组返回

当对象中有属性的enumerable被设置为false时(非枚举属性),则该属性不会在返回数组中

示例:

1.整合出对象的键 
console.log(_.keys({ a:1,b:2 }))
输出结果:
>  ["a","b"]
2.整合出数组的下标 
console.log(_.keys([5,5,5,5,5]))
输出结果:
>  [0,1,2,3,4]
3.非枚举属性不会被返回 
var o = { a:1,c:2 }
_.define(o,"b",{
  value:"?",
  enumerable:false
});

//属性已经存在但不会被枚举
console.log(o.b);
console.log(_.keys(o));
输出结果:
>  "?"
>  ["a","c"]
