_.slice

API:

_.slice(varible)

参数:

varible [ 类型:任意类型 ]

返回值:

[val,val1,...] [ 被复制的数组，或者空数组 ]

说明:

_.slice 通常用于拷贝数组,或者将非真实数组伪数组类型转化成真实的数组(如NodeList对象,Arguments对象)

如果参数不是一个数组或者不是一个类数组,则返回值均为一个空的数组

示例:

1. 拷贝数组的必要性 
var a = [1,2,3];
var b = a;
b[0] = 8;
//此时a数组的值已经发生了改变
console.log(a[0])
console.log(b[0])

var c = [1,2,3];
var d = _.slice(c);
d[0] = 9;
//被复制的数组则不会修改原数组的值
console.log(c[0])
console.log(d[0])
输出结果:
>  8
>  8
>  1
>  9
2. 参数无意义 
console.log(_.slice({}))
console.log(_.slice(""))
console.log(_.slice(321))
console.log(_.slice(null))
console.log(_.slice())
输出结果:
>  []
>  []
>  []
>  []
>  []
