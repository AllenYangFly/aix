_.splice

API:

_.splice(list,index_key,offset)

参数:

list [ 类型:Array,Object ]
index_key [ 类型:Number,String ]
offset [ 类型:Number ]

返回值:

list [ 返回参数list本身 ]

说明:

_.splice用于剔除掉数组中元素,或者删除掉对象中某个键值对

此操作直接作用参数list,执行之后list被修改

示例:

1. 删除数组中的元素，或者对象中的键值对 
var a = [1,2,3];
_.splice(a,0);
console.log(a)

var b = [3,2,"?",2];
_.splice(b,0,2);
console.log(b)

var o = { a:1,b:2 };
_.splice(o,"a");
console.log(o)
输出结果:
>  [2,3]
>  ["?",2]
>  {b:2}
