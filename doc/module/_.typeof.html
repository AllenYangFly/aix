_.typeof

API:

_.typeof(varible)

参数:

varible [ 类型:任意类型 ]

说明:

_.typeof 用于准确的检测变量类型，并且返回当前变量类型对应的字符串(小写)

类型 返回
字符串 string
数字 number
对象 object
数组 array
函数 function
函数参数 arguments
布尔值 boolean
日期 date
正则表达式 regexp
Null空 null
Undefined值 undefined
dom数组 nodelist

示例:

1. 使用typeof来检测出变量的类型 
console.log(_.typeof(1))
console.log(_.typeof("1"))
console.log(_.typeof([]))
console.log(_.typeof({}))
console.log(_.typeof(true))
console.log(_.typeof(/reg/))
console.log(_.typeof(null))
console.log(_.typeof(undefined))
console.log(_.typeof(new Date))
console.log(_.typeof(function(){}))
输出结果:
>  number
>  string
>  array
>  object
>  boolean
>  regexp
>  null
>  undefined
>  date
>  function
