_.sort

API:

_.sort(arr,func)

参数:

arr [ 类型:Array ]
func [ 类型:Function ]

返回值:

arr [ 返回被排序过的arr，或者是参数arr本身 ]

说明:

_.sort方法对数组进行排序，func参数是可选的，不传入的话则会导致js使用其默认的排序规则

方法执行成功后会立马修改arr原数组

当arr参数非法时，则会返回arr本身

示例:

1. 对数值进行由小到大排序 
var a = [31,12,1,2,23,4,15,66,3,9,84,200,8,68]
_.sort(a,function(n,m){ return n-m; })

console.log(a)
输出结果:
>  [1,2,3,4,8,9,12,15,23,31,66,68,84,200]
