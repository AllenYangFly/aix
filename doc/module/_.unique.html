_.unique

API:

_.unique(arr)

参数:

arr [ 类型:Array ]

返回值:

arr [ 返回被去重的arr，或者是参数arr本身 ]

说明:

_.unique 去重算法，将数组中的重复值剔除掉，不受到类型的影响

方法执行成功后会立马修改arr原数组

示例:

1. 去除数组中重复的元素 
var ao = { x:"?" };
var fn = function(){};
var a = [5,1,2,10,ao,4,fn,"10",4,fn,"1",6,8,4,ao,"21",1,3,fn,2,10,3,8,9,21,ao,8,fn,7,'2',4,"1"]
_.unique(a);
console.log(a)
输出结果:
>  [5, 1, 2, 10, ao, 4, fn, "10", "1", 6, 8, "21", 3, 9, 21, 7, "2"]
