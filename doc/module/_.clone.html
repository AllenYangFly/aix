_.clone

API:

_.clone(varible)

参数:

varible [ 类型:Array,Object,Number,String ]

返回值:

clone Object [ 克隆对象 ]

说明:

_.clone用于浅拷贝数组或者对象,内置使用了原生JSONAPI,

浅拷贝对象 拷贝会自动将Function类型的变量给省略掉

深拷贝可以查看 _.clonedoom

示例:

1. 拷贝对象,数组 
var fn = function(){};
var a = {a:1,b:2,c:fn};
var b = _.clone(a);
var c = [3,2,1]
var d = _.clone(c);

console.log(b===a);
//浅拷贝自动省略了Function类型的数值
console.log(b.c);
console.log(b);
console.log(c===d);
console.log(d);
输出结果:
>  false
>  undefined
>  {a:1,b:2}
>  false
>  [3,2,1]
