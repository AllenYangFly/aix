_.loop

_.foreach

API:

_.loop(list,callback(val,index,list),context)
_.foreach(list,callback(val,index,list),context)

参数:

list [ 类型:Array,Object ]
callback [ 类型:Function ]
context [ 类型:任意类型]

callback回调参数:

val [ list 每次遍历返回的值 ]
index [ list 每次遍历返回的key或者index ]
list [ 指向list本身 ]

返回值:

list

说明:

_.loop同_.foreach，用于对象或者数组的循环

示例:

1.使用循环 
var num = 0;
var list = [1,2,3,4];
_.loop(list,function(val){
  num+=val;
});

console.log(num)

var obj = { a:1,b:2,c:3 };
_.loop(obj,function(val){
  num+=val;
});

console.log(num)
输出结果:
>  10
>  16
2.叠加 index 或者 key 
var str = "call - ";

var obj = { a:1,b:2,c:3 };
_.loop(obj,function(val,key){
  str+= key + " ";
});

console.log(str)
输出结果:
>  call - a b c
3. 改变回调的this指向 
var obj = { a:1,b:2,c:3 };
_.loop(obj,function(val,key){ 
 console.log(key+this);
}," is sb");
输出结果:
>  a is sb
>  b is sb
>  c is sb
